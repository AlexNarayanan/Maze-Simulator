Maze Simulator

Generate and auto-solve mazes.

This applet generates random mazes using Kruskal's Algorithm to find a minimal spanning tree
of the grid of squares. You can automatically search through the maze with either
breadth-first or depth-first searching algorithms.

  • n- generate new maze
  • b - start new breath-first search
  • d - start new depth-first search
  • space - show/hide minimal spanning tree
  • up - generate bigger maze
  • down - generate smaller maze
